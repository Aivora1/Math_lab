Как использовать Math Lab

  • Введите уравнение в поле ввода, используя x как переменную.
  • Используйте операторы: + (сложение), - (вычитание), * (умножение), / (деление), ^ (возведение в степень).
  • Доступные функции: sin(x), cos(x), tan(x), log(x), sqrt(x), abs(x).
  • Выберите тип функции для подсказок и предустановок.
  • Установите диапазон значений по оси X для построения графика.
  • Нажмите "Построить график" для отображения функции.
  • Используйте примеры уравнений для быстрого старта.

Примечание: При вводе гиперболы (например, 1/x), график может иметь разрывы в точках, где функция не определена.
